## Install

<CodeAuthWagmiInstall/>

<Warning>
**Wagmi Compatibility**

You need to install both <InlineCode>{TxPkgAuthSDK}</InlineCode> and <InlineCode>{TxPkgAuthWagmiSDK}</InlineCode> packages. Note the version compatibility:

* For wagmi < 1.0.0 use @arcana/auth-wagmi@1.y.z
* For wagmi >= 1.0.0, use @arcana/auth-wagmi@2.y.z
* For wagmi >= 2.0.0, use @arcana/auth-wagmi@3.y.z
</Warning>

## Integrate

Use the **client ID** assigned to your app at registration and create the `AuthProvider`.

<CodeAuthCreate/>

Then use the `AuthProvider` to create a new `ArcanaConnector`.

<CodeAuthWagmiCreate/>

Next, depending upon the version of the Wagmi library used in the app, configure the `ArcanaConnector` in the Wagmi setup.

<CodeAuthWagmiConfig/>

Onboard user via the built-in plug and play login UI by using the Wagmi component in the App.

<CodeAuthWagmiOnboardPnp/>

<Info>
**Custom Login UI**

You can onboard users through a [custom login UI](/core_docs/concepts/custom_login_ui) instead of the built-in plug-and-play one. See [here](/auth/custom_login_ui) for details.

</Info>
